The point of this exercise is to design a JavaScript module that displays and hides dropdown menus. This implementation uses JavaScript to change the max-height of the clicked menu for a smooth transition. It limits the number of menus open to one at a time and moves it to the top of the Z-index so that the options aren't covered up by other menu buttons. All clicks
